# DevOps Interview Preparation Repository

This repository contains structured interview preparation materials designed for DevOps, Senior DevOps, and Lead DevOps Engineer roles.  
Content is organized using a clear folder hierarchy and topic-based files to enable systematic learning, quick revision, and real-world problem solving.

The focus is not only on tools, but also on **architecture decisions, trade-offs, reliability, security, and leadership scenarios** expected at senior and lead levels.
